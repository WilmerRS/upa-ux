<h1 align="center"> 🐣<strong> UPA UX </strong>🐯 </h1>

Extension for chrome that allows you to log into the main University vortal.


## 🐙 To get started

1. Make sure you have latest **NodeJs** version installed
2. Clone this repository: `git clone https://github.com/WilmerRS/upa-ux`.
3. Go to `upa-ux` directory run
4. Install dependencies: `yarn install`.
5. Now build the extension using: `yarn build`.
6. To avoid running `yarn build` after updating any file, you can run.

## 🐠 Adding React app extension to Chrome

In Chrome browser, go to chrome://extensions page and switch on developer mode. This enables the ability to locally install a Chrome extension.
